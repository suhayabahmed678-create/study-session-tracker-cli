# study-session-tracker-cli
A lightweight, fast, and structured Python CLI Study Tracker that lets you log study sessions, store them in JSON, and generate smart reports like topic totals and last 7-day activity.  Built with clean architecture and reusable design — perfect for learning + real use.
Features

1.Log study sessions by topic
2.Track hours spent per session
3.Automatic JSON data storage
4.Topic-wise total hours report
5.Last 7 days study summary
6.Lightweight — no external libraries required
7.Clean layered architecture
8.Beginner-friendly but production-style structure
9.Easy to extend (GUI / Web / API ready)
