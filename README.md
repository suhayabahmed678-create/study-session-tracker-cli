# study-session-tracker-cli
A lightweight, fast, and structured Python CLI Study Tracker that lets you log study sessions, store them in JSON, and generate smart reports like topic totals and last 7-day activity.  Built with clean architecture and reusable design — perfect for learning + real use.
Features

Log study sessions by topic
Track hours spent per session
Automatic JSON data storage
Topic-wise total hours report
Last 7 days study summary
Lightweight — no external libraries required
Clean layered architecture
Beginner-friendly but production-style structure
Easy to extend (GUI / Web / API ready)
